Kamox自助解封 UnbanByOneself
安装java 1.8使用双击打开。
如果不确定你的java版本请win+r输入cmd 后 输入java -version
如果不行请在当前目录下创建一个Start.cmd 内容输入
java -jar UnbanbyOneself.jar
pause
打开后请填写正确的ID 每天有五次提交申请机会。
每个计算机每天仅限提交5次;
每个ID每天仅限提交5次。
注意 如果你没被ban提交申请会消耗一次机会 没有填写正确的ID也会消耗一次 等等。
每天晚上12点重置次数。
待补充 以后会更新！
